# # 歡迎光臨 GitHub 頁。

此自動頁面產生器是最簡單的方法來創建美麗頁為所有您的專案。在這裡作者你的頁面內容 使用 GitHub 調味降價，選擇一個範本，由設計師精心設計和發佈。您的頁面生成之後，您可以簽出的新的 'gh 頁' 分支本地。如果您使用 GitHub 桌面，只需同步您的存儲庫，你會看到新的分支。

# # 設計師範本

我們已經精心設計一些帥氣的範本供您使用。向前走，然後按一下繼續佈局來流覽這些。你可以很容易再去編輯您在發佈之前的頁面。在發佈後您的頁面，你可以重新訪問頁面產生器和切換到另一個主題。你的頁面內容將被保留。

# # 手動創建頁

如果你更喜歡不使用自動產生器，推的分支命名為 'gh 頁' 你的資料庫，以手動創建網頁。除了支援常規的 HTML 內容，GitHub 頁支援傑基爾，簡單，博客知道靜態網站產生器。傑基爾很容易地創建整個網站的頁眉和頁腳，而無需將它們複製每個跨頁。它還提供智慧化的博客支援和其他高級的範本功能。

# # 作者和貢獻者

你可以 @mention GitHub 使用者名，以生成一個連結到他們的個人資料。造成 '' 的元素將會連結到參與者的 GitHub 設定檔。例如︰ 2007 年，克裡斯 Wanstrath (@defunkt)，PJ Hyett (@pjhyett) 和湯姆普勒斯頓 · 沃納 (@mojombo) 成立 GitHub。
